# Topic brief — B-AB1

**Track:** BUILD / ABSORB 1
**Working title:** Build With AI — Specs, Agents & the Build Loop
**Audience:** HGDW members starting the BUILD track. Smart, ambitious, not (yet) engineers.
**Goal of this lesson:** By the end, the viewer can (1) write a spec an agent can execute,
(2) understand the agent build loop, and (3) run their first agent-driven build.

**Teaching arc:** SOURCE (why specs beat prompts) → ABSORB (the build loop) →
MIRROR (read a real spec + run it) → COMMAND (their first build + the correction loop).

**Tone:** warm, direct, a little irreverent. No fluff. High-impact systems, not busywork.

**Must include:**
- The line: "A prompt is a wish. A spec is an instruction."
- A short terminal demo (DO) of kicking off a build.
- A compare frame: prompting vs. spec-driven.
- A closing CTA to the next lesson.
