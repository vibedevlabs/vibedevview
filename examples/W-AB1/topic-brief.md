# Topic brief — W-AB1

**Track:** WORK / ABSORB 1
**Working title:** Mastering Team File Management with Claude
**Audience:** HGDW members and workspace collaborators managing shared projects. Comfortable with tools, not necessarily engineers.
**Goal of this lesson:** By the end, the viewer can (1) explain how Claude Cowork and Claude Code replace manual file management, (2) set up a sandboxed workspace with proper context files, and (3) run an agentic organization pass on a real project directory.

**Teaching arc:** SOURCE (the death of drag-and-drop) → ABSORB (three key features: shared context, agentic org, admin controls) →
MIRROR (three best practices with a context-file example) → COMMAND (live demo of Claude cleaning a messy directory).

**Tone:** warm, direct, a little irreverent. No fluff. High-impact systems, not busywork.

**Must include:**
- The hook: manual file management is dead.
- Three features: Shared Context Hub, Agentic Organization, Team Admin Controls.
- Three best practices: sandbox, naming conventions, context files (.claude/rules/, CLAUDE.md).
- A code frame showing a context-file directory structure.
- A DO segment for a live Claude cleanup demo.
- A closing CTA to try it on their own project.
